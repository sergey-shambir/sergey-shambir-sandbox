Simple plugin for kawai Python highlighting.

If you want to compile this plugin, you should:

  • Setup Qt build enviroment (mingw + qmake + Qt for Windows or gcc + qt-devel + qmake for Linux)

  • Download QtCreator source code here: qt.nokia.com/downloads

  • put folder with source code ("pythoneditor") into "%QT-CREATOR-SOURCE%/src/plugins"

  • modify ".../src/plugins/plugins.pro" file like this:


    SUBDIRS = plugin_coreplugin \
              # ...
              plugin_welcome \
              plugin_pythoneditor
    # ...
    # at file end:
    plugin_pythoneditor.subdir = pythoneditor
    plugin_pythoneditor.depends = plugin_texteditor

  • run "qmake qtcreator.pro && make"

  • Look in "bin" folder. Enjoy Python coding!
